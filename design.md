## Colors

- Deep blue: `#13315c`
- Forest green: `#0b6e4f`
- Gold accent: `#f4d35e`
- Soft page background: `#f7f9fb`
- Text color: `#1f2933`
- Light border: `#d6dee6`

## Logo

The logo is saved as `images/portal-logo.svg`. It uses a compass shape to represent learning direction and the initials `S V`.

## Fonts

- Headings: Georgia, Times New Roman, serif
- Body text: Arial, Helvetica, sans-serif

## Layout Notes

The page uses semantic HTML elements: `header`, `nav`, `main`, `section`, `article`, and `footer`. CSS Grid is used for the hero, overview cards, and ICE challenge cards.
